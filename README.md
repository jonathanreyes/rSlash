# rSlash
Opera/Chrome omnibox extension to quick-nav to a particular subreddit, using "r" as the keyword <br />

To Use: <br />
Type "r \<sub\>" into the omnibox and hit enter to navigate to reddit.com/r/\<sub\> <br />
